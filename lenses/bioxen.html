Core Research Question

How can we implement a production-grade four-lens signal analysis system (Fourier/Lomb-Scargle, Wavelet, Laplace, Z-Transform) for biological temporal dynamics in synthetic cells, leveraging mature Python libraries while meeting peer-reviewed research standards?

Guiding Philosophy

Write minimal integration code (~450 lines) to leverage the power of proven, heavily tested libraries (~500,000+ lines), ensuring reliability and maintainability.

Mandatory Validation Checks

Nyquist Criterion

Sampling rate must be at least 2x the maximum frequency to avoid aliasing.

Signal Quality

Check for minimum data length, sufficient variance, and acceptable Signal-to-Noise Ratio (SNR).

Data Integrity

Detect and handle constant signals, NaN values, and significant outliers before analysis.

Detrending

Determine if linear, polynomial, or spline detrending is required to isolate oscillations.

Biological Use Cases

This section grounds the analytical methods in practical biological problems. The four-lens framework is designed to address diverse challenges, from detecting stable daily rhythms to capturing transient stress responses and engineering synthetic circuits.

Circadian Rhythm Detection

How can we detect a ~24h periodicity in gene expression with statistical significance? This requires robust methods like Lomb-Scargle for noisy, irregularly sampled data.

Cell Cycle Analysis

How to track non-stationary phase transitions (G1‚ÜíS‚ÜíG2‚ÜíM)? Wavelet analysis is essential here, as it can pinpoint frequency changes over time, something Fourier analysis misses.

Stress Response & Damping

How to detect and characterize transient events like heat shock? Wavelets excel at finding sharp, temporary signals, while tools like PAICE/ECHO analyze the subsequent damping oscillations.

Feedback Control Design

How can Laplace-domain transfer functions be used to design stable genetic circuit controllers? This involves control theory methods to ensure synthetic biological systems behave as intended.
